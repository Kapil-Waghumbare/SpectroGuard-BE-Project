# SPECTROGUARD: AI Powered Cyber Threat Detection
> A web-based cybersecurity toolkit for detecting phishing, weak passwords, and unsafe URLs.
---
## 📌 Overview

SpectroGuard is an AI-based cybersecurity toolkit that combines multiple security tools into one platform.
It helps users:
- Detect phishing websites
- Check password strength
- Analyze suspicious URLs
- Generate secure hash IDs
The system uses Machine Learning models and heuristic techniques to detect real-world cyber threats.
---

## ❗Problem Statement

- Increasing cyber attacks (phishing, weak passwords)
- Users unaware of online threats
- Tools are complex and scattered
---

## 💡 Solution

SpectroGuard provides:
- One platform with multiple tools
- AI-based detection system
- Simple UI for non-technical users
- Privacy-focused (no data storage)

---

## ✨ Features

- 🔐 Password Strength Checker  
- 🌐 Phishing Website Detector  
- 🔗 URL Safety Checker  
- 🔑 Hash ID Generator  
- ⚡ Fast and simple interface  
- 🔒 No user data stored  

---

## 🧰 Tools

### 1. Password Strength Checker
- Uses Higher-Order Markov Model
- Detects weak vs strong passwords

### 2. Phishing Website Detector
- Uses SVM (Support Vector Machine)
- Classifies website as Safe or Phishing

### 3. URL Detector
- Blacklist + Whitelist + Heuristic Analysis

### 4. Hash Generator
- Uses SHA-256 algorithm
- Generates secure hash
