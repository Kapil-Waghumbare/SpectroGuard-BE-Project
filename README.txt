HOW TO RUN SPECTROGUARD
========================


LINUX / KALI (First Time Only)
--------------------------------
1. Open terminal
2. cd "path/to/Project SpectroGuard/Web"
3. python3 -m venv venv
4. source venv/bin/activate
5. pip install flask flask-cors scikit-learn pandas numpy


LINUX / KALI (Every Time)
--------------------------
1. Open terminal
2. cd "path/to/Project SpectroGuard/Web"
3. source venv/bin/activate
4. python app.py
5. Double click Index.html in the Web folder


----------------------------------------------------------------


WINDOWS (First Time Only)
---------------------------
1. Open Command Prompt
2. cd "path/to/Project SpectroGuard/Web"
3. python -m venv venv
4. venv\Scripts\activate
5. pip install flask flask-cors scikit-learn pandas numpy


WINDOWS (Every Time)
----------------------
1. Open Command Prompt
2. cd "path/to/Project SpectroGuard/Web"
3. venv\Scripts\activate
4. python app.py
5. Double click Index.html in the Web folder


----------------------------------------------------------------

NOTE: Keep the terminal/command prompt open while using the website.
