What does this analyzer do?

This tool runs your input through SpectroGuard's algorithm and a comparison algorithm side by side. It measures how well each algorithm performs across multiple metrics — so you can see exactly where ours is stronger, and where the trade-offs are. All computation happens live in your browser.
